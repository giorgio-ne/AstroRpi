# AstroPi
Script to setup Astropy and Jupyter Notebook on Linux

$ chmod +x AstroRpi
$ sudo ./AstroRpi
$ source /home/pi/jupyter-env/bin/activate
$ jupyter notebook --generate-config
$ deactivate
$ chmod +x notebookConf
$ ./notebookConf

Startup AstroRpi with:
$ cd astropy-notebook
$ ./run-jupyter-notebook.sh &
